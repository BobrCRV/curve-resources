<h1>Mutli-Chain: Curve DAO Token</h1>

The Curve token **can be bridged** across some chains, but does **not always have full functionality**. Locking of CRV for veCRV must be done on Ethereum. Rewards voting for cross-chain gauges occurs on Ethereum aswell.
